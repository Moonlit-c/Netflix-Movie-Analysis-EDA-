# Netflix-Movie-Analysis-EDA
This project explores the Netflix movies dataset to analyze content distribution, genre trends, and production patterns. Using Python for data cleaning and exploratory analysis, and Tableau for visualization, the project identifies key insights about Netflix’s movie catalog.

Tools used: 
  * Python (Matplotlib, Pandas, Seaborn)
  * Jupyter Notebook
  * Tableau

Dataset: The dataset contains information about Netflix movies including title, genre, release date, votes, original language, overview and popularity.
Rows: ~10000

Key Questions Answered:
 1. What is the most frequent genre of movies released on Netflix?
 2. Which has the highest votes in the vote average volumn?
 3. What movie has the highest popularity? What is its genre?
 4. What movie has the lowest popularity? What is its genre?
 5. Which year has most filmed movies?

Key Insights:
 1. Drama genre is the most frequent genre in our dataset.
 2. Mostly average movies have highest number of votes.
 3. Spiderman: No Way Home has the highest popularity whose genres are Action, Adventure and Science Fiction
 4. The United States vs. Billie Holiday has lowest popularity with genres Music, Drama, History as well as Threads with genres War, Drama, Science Fiction
 5. 2020 has the most number of movies filmed.
